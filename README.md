# GOApostille.com

Static landing page project for GOApostille.com, built for Vercel.

## Pages

- `/` - homepage jasa apostille, legalisasi dokumen, dan penerjemah tersumpah
- `/apostille-dokumen-pernikahan/` - landing page apostille/legalisasi buku nikah dan akta nikah
- `/apostille-dokumen-pendidikan/` - landing page apostille/legalisasi ijazah dan transkrip nilai
- `/jasa-penerjemah-tersumpah/` - landing page penerjemah tersumpah Jakarta
- `/en/` - English homepage
- `/en/marriage-document-apostille/` - English marriage document apostille page
- `/en/education-document-apostille/` - English education document apostille page
- `/en/sworn-translator-jakarta/` - English sworn translator landing page

## Local workflow

```bash
npm run build
npm run check
npm run serve
```

## Deploy

Connect this repository to Vercel as a static project. No build command is required if generated files are committed, but `npm run build` is safe to run before deploy.
